A Paixão Pela Cultura Automotiva

O Gearhead Project nasceu da vontade de reunir os melhores projetos automotivos em um só lugar. Uma vitrine para quem respira graxa, potência e customização.

Sobre o Desenvolvedor

Este projeto foi desenvolvido por Victor Dourado como Trabalho Prático para o curso da PUC Minas. O objetivo é demonstrar habilidades em Front-End, consumo de APIs REST com JSON Server, manipulação do DOM e criação de dashboards interativos.
